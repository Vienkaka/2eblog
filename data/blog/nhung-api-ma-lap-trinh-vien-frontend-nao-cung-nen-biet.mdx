Lập trình viên Frontend vừa phải tạo UI vừa phải giao tiếp với API để show dữ liệu và cho phép người
dùng tương tác với website. Vì vậy để tăng tốc độ làm việc và có hiệu quả hơn thì cần có những thư viện,
APIs giúp hỗ trợ cho lập trình viên có thể đẩy nhanh tốc độ công việc mà giảm sự phụ thuộc vào những
thành viên liên quan. Dưới đây là một số APIs mình muốn giới thiệu tới các bạn:

## JSONPlaceholder

![](https://images.viblo.asia/0dd845e4-c6f9-443e-ae51-d0282bb09a34.png)
API này rất phổ biến với các lập trình viên. Thông thường thì API này dùng cho việc định nghĩa type, đổ dữ liệu vào UI giúp lập trình viên Frontend thấy được UI của mình với lượng dữ liệu lớn để testing.

### Tính năng:

- Không yêu cầu đăng ký tài khoản
- Không cần bất kì cài đặt nào
- Giúp tạo API một cách tự động
- Chia sẻ bất kỳ dữ liệu nào
- Hỗ trợ các method như: POST, PUT, ....

## Fake Store API

![](https://xlafgkujgoaelkhxzzpo.supabase.co/storage/v1/object/public/affiliate/cource/fackapi.svg)
Được sử dụng phần lớn cho các trang thương mại điện tử, các trong mua sắm online. Nó cung cấp dữ liệu giả cho các ứng dụng mua sắm online. Fake Store API cung cấp api reatime giúp ứng dụng đi đến giai đoạn testing ngay mà không cần phải run phía server. Hơn thế nữa, Fake Store API cung cấp các method GET, POST, PUT,... chuẩn refulAPI giúp lập trình viên có thể phát triển bất kỳ ứng dụng nào.

### Tài nguyên cung cấp:

- Sản phẩm
- Giỏ hàng
- Người dùng
- Authentication

## Unsplash API

![](https://xlafgkujgoaelkhxzzpo.supabase.co/storage/v1/object/public/affiliate/cource/unsplash.webp)
Là api cung cấp dữ liệu lớn các hình ảnh, nó cung cấp tới hơn 600 ngàn hình ảnh chất lượng free. Các lập trình viên sử dụng chúng thoải mái mà không phải chi trả 1 đồng nào:

### Tính năng:

- Hình ảnh có thể sử dụng cho cả cá nhân và doanh nghiệp
- Dễ dàng sử dụng, dễ dàng setup
- Nhanh và linh hoạt

### Mức độ phổ biến

- 42 triệu lượt tải/ tháng
- 8 tỷ hình ảnh sử dụng/ tháng
- 16 hình ảnh được tải xuống/ giây
