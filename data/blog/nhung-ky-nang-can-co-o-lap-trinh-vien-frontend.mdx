Có nhiều mãng lập trình viên khác nhau, có người làm coding, có người bảo trì, người thì xuất dữ liệu vv. Trong khi lập trình viên Backend tạo các kiến trúc để website gởi request và nhận về response thì lập trình viên Frontend tập trung vào trải nghiệm người dùng.

## 1. Kỹ năng cơ bản

Lập trình Frontend phải có những kỹ năng cơ bản là xây dựng các tính năng được đánh giá cao và mang lại bảo mật cho websites. Có 1 vài công nghệ mà bất kỳ 1 lập trình viên Frontend nào cũng phải viết là:

- HTML
- CSS
- Sass
- Javascript/ Typescript
- DOM manipulation

## Framworks

Kiến thức căn bản rất quan trọng nhưg Frameworks sẽ giúp tốc độ phát triển ứng dụng của bạn nhanh hơn và triển khai dễ dàng hơn

- Bootstrap
- Taiwind CSS
- React
- Vue
- NextJs

## Kỹ năng thiết kế

Khi mà việc code trang web rất quan trọng thì website cần phải chạy trên cả Laptop, Tablet và điện thoại. Vì vậy bạn cần phải có những bản thiết kế UI/UX cho nhiều loại thiết bị khác nhau. 1 vài kỹ năng giúp bạn đứng vững:

- Canva
- Figma
- Photoshop
- Color

## Quản lý thời gian

Khi đang là một lập trình viên mới vào nghề nhưng bản phải có sự quan tâm và sẳng sàng để trờ thành 1 lập trình viên Pro. Quản lý thời gian là kỹ năng không thể bỏ qua. Bạn phải tính toán và sử dụng thời gian của bạn một cách hợp lý:

- Tạo danh sách công việc
- Sử dụng Promodoro Clock
- Sử dụng các công cụ quản lý thời gian như MS Excel

## Sử dụng Web tools

Việc tìm tòi và sử dụng các công cụ hỗ trợ cho việc tạo dựng web giúp bạn tối ưu thời gian, dễ dàng bắt lỗi và sửa lỗi:

- Imporve CSS and Javascript
- Code Generators
- Code Optimize tools
- Code Validation

## Testing

Testing giúp bạn hạn chế nhất lỗi đến với người dùng. Giúp bạn chuẩn chỉnh trong lúc code và giúp hạn chế các bên tham gia để task đi một cách sớm nhất và hiệu quả nhất:

- Test hiệu năng
- Tets với vai trò người dùng cuối
- Test trên các nền tảng khác nhau

## Kỹ năng mềm

Kỹ năng cộng đồng cũng rất quan trọng, giúp bạn dễ dàng học hỏi và nâng cao tay nghề 1 cách nhanh chóng

- Kỹ năng lắng nghe
- Ký năng đọc
